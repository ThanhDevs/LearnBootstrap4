## Nôi dung chính

* w-*
* h-*

Website Tham khảo：

https://getbootstrap.com/docs/4.0/utilities/sizing/

## Bài tập thực hành

~~~html
<style>
    .my-container {
        width: 200px;
        height: 200px;
        background: #ddd;
        margin-bottom: 5px;
    }
</style>

<div class="my-container">
    <div class="bg-primary w-25 h-50"> ThanhDevs Chanel </div>
</div>
<div class="my-container">
    <div class="bg-success w-75 h-100"> ThanhDevs Chanel </div>
</div>
~~~
